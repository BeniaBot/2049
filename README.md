# 2049  —  מהדורה משוכללת של 2048

גרסה משודרגת של 2048 בעיצוב הקלאסי, עם **מקשי עזרה נסתרים** הניתנים להגדרה.
כל עזרה נראית בדיוק כמו מהלך רגיל — אין שום סימן חיצוני — כך שאפשר לשחק מול
קהל בזמן שהמקשים הנסתרים עוזרים לך בשקט.

עברית כברירת מחדל · אנגלית בהגדרות · עיצוב 2048 הקלאסי · אנימציית החלקה
אמיתית · רינדור חד · צלילים עם **טון עולה לפי ערך האריח** · בורר גודל לוח
מהיר · מצב אינסופי/שיא · אייקון מותאם · **בדיקת עדכונים אוטומטית** · אופליין.

> **הגדרות ושיאים** נשמרים ב-`%APPDATA%\2049\` — גם אם תמחק קבצים ליד ה-exe,
> הם לא ייפגעו.

---

## הפיכה ל-exe עצמאי (אופליין)

פעם אחת, על מחשב Windows עם Python מותקן:

1. התקן Python 3.9+ מ-python.org (סמן "Add to PATH").
2. שים את כל הקבצים בתיקייה אחת.
3. לחיצה כפולה על **`build_exe.bat`**.
4. המשחק נמצא ב-**`dist\2049.exe`** (עם אייקון מותאם).

ה-exe עצמאי לחלוטין: רץ על כל מחשב Windows בלי שום דבר מותקן.

## איך משחקים

- הזזת אריחים עם **מקשי החצים** או **WASD**.
- **משחק נגמר?** לחץ **Enter** למשחק חדש.

### כפתורים במסך המשחק
- **בורר `4×4`** (מימין למעלה) — חצים להחלפת גודל הלוח, ואייקון ⟳ למשחק חדש.
- **הגדרות** (משמאל למעלה). *אין תפריט — יציאה = X של החלון.*
- **🔊 קול** ו-**∞/🎯 מצב** (בתחתית).

> **האייקון** (לוח 2049) מופיע ב-exe ובשורת המשימות אחרי קימפול עם
> `build_exe.bat`. מ-Python ישיר ווינדוס עשוי להציג אייקון ברירת מחדל.

## הגדרות ו"מקלט הרמאים" 😏

מסך **ההגדרות** מכיל דברים תמימים: גודל לוח, שפה, קול, מצב ניצחון. אם מישהו
פותח אותו מול קהל — לא רואים כלום חשוד.

מנגנון הרמאות מוסתר בחלון בשם **"מקלט הרמאים"** — נפתח דרך כפתור קטן בתחתית
ההגדרות. *"פה מתאספים כל מי שלא מספיק טובים בזכות עצמם."* 😄

**טוויסט:** כדי לשמור הגדרות רמאות, המשחק ידרוש ממך **להודות** שאתה מרמה כי
אתה לא מספיק טוב. אם תסרב — ההגדרות לא יישמרו והמשחק "ישלח אותך להתאמן".

**פירוט מלא של כל אפשרויות הרמאות: `מדריך_הרמאות.md`.**
