# איך להעלות את 2049 למאגר שלך — הוראות מדויקות

המאגר שלך: **https://github.com/BeniaBot/2049**

יש לך שתי אפשרויות. הראשונה (לחיצה אחת) היא הכי קלה.

---

## אפשרות א' — בלחיצה אחת (מומלץ)

1. ודא שהמאגר `https://github.com/BeniaBot/2049` **קיים וריק**
   (בלי README, בלי .gitignore). אם כבר יצרת אותו — מצוין.
2. חלץ את הזיפ, ותכנס לתיקיית `2049`.
3. **לחיצה כפולה על `upload_to_github.bat`** (כבר מוגדר למאגר שלך).
4. אם git מבקש התחברות — ייפתח חלון GitHub, או שתתבקש להדביק Personal
   Access Token (ראה בהמשך).
5. זהו. רענן את דף המאגר.

---

## אפשרות ב' — ידנית, צעד-אחר-צעד

פתח **PowerShell** או **Command Prompt** *בתוך* תיקיית `2049`:
- דרך קלה: בתוך התיקייה, Shift + קליק-ימני על אזור ריק →
  "Open PowerShell window here" (או "פתח חלון PowerShell כאן").

עכשיו הקלד את הפקודות הבאות, אחת-אחת (Enter אחרי כל שורה):

```
git init
git branch -M main
git add .
git commit -m "2049 - enhanced 2048 with hidden assists"
git remote add origin https://github.com/BeniaBot/2049.git
git push -u origin main
```

מה כל פקודה עושה:
- `git init` — יוצר מאגר git מקומי בתיקייה.
- `git branch -M main` — קובע את הענף הראשי בשם main.
- `git add .` — מוסיף את כל הקבצים.
- `git commit -m "..."` — יוצר "צילום מצב" עם תיאור.
- `git remote add origin ...` — מקשר לתיקייה המרוחקת (המאגר שלך).
- `git push -u origin main` — מעלה הכל לגיטהאב.

## אם git מבקש סיסמה / טוקן

גיטהאב לא מקבל סיסמה רגילה ב-push. אם תתבקש "Password":

1. היכנס ל-https://github.com/settings/tokens
2. **Generate new token → (classic)**
3. תן שם, בחר תוקף, וסמן את ההרשאה **`repo`**.
4. **Generate token** והעתק אותו (מוצג פעם אחת בלבד!).
5. כשה-push מבקש "Password" — הדבק את הטוקן.

הטוקן יישמר ב-Windows Credential Manager ולא תצטרך אותו שוב.
> את הטוקן מזינים רק במחשב שלך, בחלון git. אין צורך לשלוח אותו לאף אחד.

## עדכונים עתידיים (ושחרור גרסה חדשה)

### כדי להעלות שינויים בקוד:
```
git add .
git commit -m "תיאור השינוי"
git push
```

### כדי שהמשחק יזהה עדכון אוטומטית ויוריד אותו ישירות:
1. עדכן את מספר הגרסה בקוד — בקובץ `2049.py`, בשורה:
   `APP_VERSION = "0.9.0"` → למשל `"1.0.0"`.
2. בנה מחדש את ה-exe עם `build_exe.bat` (נוצר `dist\2049.exe`).
3. עשה commit + push לשינוי הקוד.
4. בגיטהאב: לך ל-**Releases → Draft a new release**.
5. ב-"Tag" כתוב גרסה עם v, למשל `v1.0.0` (חייבת להיות **גבוהה**
   מהגרסה הקודמת).
6. **גרור את `dist\2049.exe`** לאזור הקבצים בתחתית.
   ⚠️ **חשוב: הקובץ חייב להישאר בשם `2049.exe`** — ככה כפתור "הורד גרסה
   חדשה" במשחק יוריד אותו ישירות (הקישור הוא
   `.../releases/download/<tag>/2049.exe`).
7. **Publish release**.

מעכשיו, כל מי שמריץ גרסה ישנה יותר יראה "גרסה חדשה זמינה" במשחק, וכפתור
**"הורד גרסה חדשה"** יוריד את ה-exe ישירות במחשב שלו.
